Что я умею делать?

Первичная настройка linux серверов

Настройка различных конфигурация linux серверов. Таких как сервер электронной почты, прокси сервер, vpn сервер, сетевой экран, хостинг сервер, dns сервер, сервер видеонаблюдения, файловый сервер, игровой сервер и т.д.

Администрирование linux серверов

Удалённое администрирование различных конфигураций linux серверов. Обновление и backup софта, мониторинг работоспособности, оперативное устранение неисправностей и прочее.

Виртуализация серверов

Настройка сервера виртуализации и перенос действующих серверов в виртуальную среду. Настройка отказоустойчивых кластеров виртуализации и прочие услуги.

Веб разработка ( php+mysql )

Разработка и сопровождение различных веб приложений на основе php+mysql+bootstrap.

Выполненые проекты

Настройка почтового сервера

Выполнена настройка почтового сервера для АО "НПФ "СНГ". Настройка сделана по наиболее надёжной и распространённой схеме Debian, Postfix, Courier, Saslauth, MySQL. Фильтрация писем реализована при помощи связки amavisd-new, SpamAssassin и ClamAV. В качестве веб интерфейса для работы с почтой использован Web Mail Lite PHP. Дополнительно было написано веб приложение для отображения адресной книги предприятия.

Срок: 2 дня. Цена: 10 000 рублей

Виртуализация серверов

Выполнена настройка сервера виртуализации и перенос в виртуальную среду серверов рекламного агентства "Динамика Света". В качестве гипервизора был использован xen. Были завиртуализированны три сервера компании: сервер видеонаблюдения, веб сервер, сервер базы данных. Так же был реализован автоматический backup на отдельное хранилище находящиеся физически в другом здание.

Срок: 5 дней. Цена: 30 000 рублей.

Разработка веб приложения

Выполнено проектирование и разработка веб приложения для ООО «СМИА СИА-ПРЕСС». Написанно на языке PHP, извользована база данных mysql, интерфейс основан на css фреймворке bootstrap и jquery. Данное приложение включает в себя средство администрирования пользователей системы, формы добавления и редактирования объявлений, формирование плейлистов, автоматическое формирование счетов, выгрузку отчётов в формате xls.

Срок: 1.5 месяца. Цена: 50 000 рублей

Настройка сервера видеонаблюдения

Разработка веб приложения

На основе тарифного руководства по продукту "ОСАГО" была спроектирована и разработана фронт офисная система. Данная система позволяет агентам удалённо, через веб интерфейс, производит расчёт и оформление полисов "ОСАГО". Так же реализовано администрирование пользователий, филиалов и бланков строгой отчётности в рамках данного продукта. Веб интерфейс продукта реализован при помощи css фреймворка bootstrap, что в итоге позволяет отображать корректно информацию в независимости от разрешения экрана и устройства конечного пользователя системы. Так же в рамках данного проекта была реализована выгрузка данных из базы данных mysql в базу данных Oracle, sms информирование пользователей. Печатные формы формируются в формате docx и pdf.
