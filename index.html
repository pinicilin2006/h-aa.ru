Первичная настройка linux серверов

Настройка различных конфигурация linux серверов. Таких как сервер электронной почты, прокси сервер, vpn сервер, сетевой экран, хостинг сервер, dns сервер, сервер видеонаблюдения, файловый сервер, игровой сервер и т.д.

Администрирование linux серверов

Удалённое администрирование различных конфигураций linux серверов. Обновление и backup софта, мониторинг работоспособности, оперативное устранение неисправностей и прочее.

Виртуализация серверов

Веб разработка ( php+mysql )

Разработка и сопровождение различных веб приложений на основе php+mysql+bootstrap.

Настройка почтового сервера

Выполнена настройка почтового сервера для АО "НПФ "СНГ". Настройка сделана по наиболее надёжной и распространённой схеме Debian, Postfix, Courier, Saslauth, MySQL. Дополнительно было написано веб приложение для отображения адресной книги предприятия.

Срок: 2 дня. Цена: 10 000 рублей

Разработка веб приложения

Выполнено проектирование и разработка веб приложения для страхового общества "Сургутнефтегаз". Написанно на языке PHP, извользованна база данных mysql, интерфейс основан на css фрэймворке bootstrap и jquery. На основе тарифного руководства по продукту "ОСАГО" был спроектированна и разработанна фронт офисная система. Данная система позваляет агентам удалённо, через веб интерфейс, произовдит расчёт и оформление полисов "ОСАГО". Так же реализованно администрирование пользователий, филиалов и бланков строгой отчётности в рамках данного продукта. Веб интерфейс продукта реализован при помощи css фрэймворка bootstrap, что в итоге позволяет отображать корректно иинформацию в независимости от разрешения экрана и устройства конечного пользователя системы. Так же в рамках данного проекта была реализованна выгрузка данных из базы данных mysql в базу данных Oracle, sms информирование пользователей. Печатные формы формируются в формате docx и pdf. Срок: 2.5 месяца. Цена: 150 000 рублей
